# PDF2KG 版本历史

## V0 - 基础版本 (当前)

### 功能
- ✅ PDF 文本提取
- ✅ 文本分块处理
- ✅ DeepSeek API 提取实体和关系
- ✅ 多线程并发处理
- ✅ 磁盘缓存
- ✅ NetworkX 可视化
- ✅ Neo4j 存储

### 已知问题
- 实体提取不够准确
- 关系类型过于简单
- 图谱结构松散
- 缺少实体消歧
- 缺少关系验证

### 输出
- `output/kg_batch.json` - 知识图谱数据
- `output/kg_batch.png` - 可视化图片 (V0)
